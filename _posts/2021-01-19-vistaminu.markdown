When we first set out to buy a stroller for our first baby, we couldn not find too many firsthand comparison reviews between the Uppababy Vista and UPPAbaby Minu. We first bought the Vista, then the Minu, and have some experience using the two and have opinions on which we would recommend to new parents, so here is a video summarizing our thoughts.

<iframe width="716" height="402" src="https://www.youtube.com/embed/96IzT05njq8" frameborder="0" allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture" allowfullscreen></iframe>
<br />

Overall, I think if we were to go back in time and pick only one stroller we would choose the Minu. We like the stroller more because it has a smaller foot print and it much more portable. The weight is a plus when traveling, but it is also a big plus for everyday use.
